# 🚀 Esercizi Base in Linguaggio C

Benvenuto in questo repository! Qui troverai una raccolta di esercizi fondamentali per iniziare a programmare in **C**. Il materiale è pensato per chi sta muovendo i primi passi con la logica di programmazione, le variabili e le strutture di controllo.

## 📁 Contenuto del Repository

Il progetto contiene un archivio compresso principale:
* **`esC.zip`**: Una cartella zippata che racchiude diversi file sorgente `.c`, ognuno dedicato a un concetto specifico della programmazione procedurale.

### Argomenti Trattati
Gli esercizi all'interno della cartella `esC` coprono i seguenti pilastri:
1.  **Input/Output**: Uso corretto di `printf` e `scanf`.
2.  **Operazioni Aritmetiche**: Somma, sottrazione, moltiplicazione, divisione e calcolo del resto (modulo).
3.  **Strutture Condizionali**: Gestione dei flussi con `if`, `else` e `switch`.
4.  **Cicli e Iterazioni**: Utilizzo di cicli `for`, `while` e `do-while`.
5.  **Array e Stringhe**: Manipolazione di vettori di numeri e array di caratteri.

---

## 🛠️ Come Utilizzare gli Esercizi

1.  **Scarica il repository**: Scarica il file ZIP o clona il repository sul tuo computer.
2.  **Estrai l'archivio**: Estrai il contenuto di `esC.zip` in una cartella locale.
3.  **Compilazione**: Utilizza un compilatore come `gcc`. Apri il terminale nella cartella estratta e digita:
    ```bash
    gcc nome_esercizio.c -o nome_esercizio
    ```
4.  **Esecuzione**: Avvia il programma compilato:
    ```bash
    ./nome_esercizio
    ```

---

## 📝 Prerequisiti

Per poter compilare ed eseguire il codice, avrai bisogno di:
* Un compilatore C (es. **GCC**, **Clang** o quello integrato in un IDE come Code::Blocks).
* Un editor di testo (VS Code, Sublime Text) o un ambiente di sviluppo completo.
